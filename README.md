# Intermediate Answer Keys Website

A modern, mobile-friendly website for students to access and download answer keys for different subjects.

## Features

- ✅ Student login and signup system using **Firebase Authentication**
- ✅ Admin panel to upload **PDFs, images, and answer keys daily**
- ✅ Homepage showing **latest uploaded answers**
- ✅ Subject categories: **Mathematics, Physics, Chemistry, Biology, etc.**
- ✅ Search bar to quickly find answers
- ✅ **Mobile-friendly and responsive design**
- ✅ Built with **HTML, CSS, JavaScript**, and **Firebase**

## How It Works

1. Students sign up or log in using email and password.
2. Admin can upload new answers through Firebase.
3. Students can view or download answers organized by subject.
4. All uploaded content is visible in the **latest answers section** on the homepage.

## Setup Instructions

1. Clone the repository or download the code.
2. Go to **Firebase** → create a **Web App** → copy the Firebase config.
3. Paste the Firebase config into `index.html` inside the `<script>` tag.
4. Open the website in a browser to test signup/login functionality.
